# Проект на Хакатон ВТБ MoreTech 4.0
Данное WEB-приложение создавалось с целью привлечения молодых специалистов в штат сотрудников ВТБ, а также геймификации процесса работы. Приложение использует функции создания и распространения NFT, а также транзакции с использованием электронной валюты.

# Запуск Frontend:
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).

#Запуск Backend:
Запустить файл main.py
