# Проект на Хакатон ВТБ MoreTech 4.0
Данное WEB-приложение создавалось с целью привлечения молодых специалистов в штат сотрудников ВТБ, а также геймификации процесса работы. Приложение использует функции создания и распространения NFT, а также транзакции с использованием электронной валюты. В этом репозитории содержится только backend часть приложения, над разработкой которой я принмал участие.
